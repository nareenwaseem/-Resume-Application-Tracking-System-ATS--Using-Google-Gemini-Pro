#  Resume Application Tracking System(ATS) Using Google Gemini Pro
 
ATS Resume Expert: An interactive Streamlit app that leverages Google Generative AI to analyze resumes and job descriptions. Users can upload PDFs, receive detailed evaluations on job match percentage, identify strengths and weaknesses, and get tailored improvement tips. Key features include skill and keyword analysis, experience relevance assessment, and achievements review. The app transforms resumes into images for AI analysis and provides a chatbot feature for additional insights.
